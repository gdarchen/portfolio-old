Erneurung die Webapplikation „Stratow“ für AEROW Solutions

INSA Rouen-Normandie (76)• Jan. 2017 — Jan. 2018

Halbzeitprojekt bei der INSA Rouen-Normandie für die Firma AEROW Solutions realisiert. In einem Team von acht Schülern soll dieses ISO 9001:2015 zertifizierte Projekt den Übergang von der Webplattform Electronic Document Management zu MVC (Model-View-Controller) mit dem Symfony-Framework sowie die komplette Erneurung von Die Front-End-Architektur (dank Bootstrap-Framework). Wir entwickelten in TDD (Test Driven Development) und die Arbeitsmethodik ist eine Agile Methode namens Scrum.
Während der ersten 6 Monate war ich Projektleiter.


Entwicklung einer Instant Messaging-Anwendung : „WhatsASI?“

INSA Rouen-Normandie (76)• Jan. 2017 — Apr. 2017

In Java 8 und mit der RMI-Technologie als Teil eines Distributed Computing Kurses, die Erstellung einer Instant Messaging-Anwendung, so dass der Benutzer von verschiedenen Maschinen chatten zu ermöglichen. Benutzer können einen Pseudo, ein Profil-Avatar wählen und entscheiden, in welcher Konversation sie chatten möchten. Es ist auch möglich, mit einer Künstlichen Intelligenz (KI) zu plaudern.
Es stehen zwei grafische Schnittstellen zur Verfügung : der erste wurde dank JavaFX entwickelt, der andere wurde in einem Terminal eingesetzt.

Entwicklung eine Anwendung um den Bildschirm umscharf zu machen

INSA Rouen-Normandie (76)• Sept. 2016

In Python mit der OpenCV-Programmbibliothek, die Entwicklung einer Anwendung, die es dem Benutzer ermöglicht, seinen Computer zu benutzen und das von der Webcam aufgenommene Bild anzuzeigen, sobald eine Person hinter ihm erscheint. So zeigt auf dem angezeigten Bild ein grüner Rahmen den Ort des neu erkannten Gesichts an, damit der Benutzer wissen kann, wer ihn ausspioniert.
Es gibt zwei andere Modi wie die vorgestellt: ein „Zen“-Modus, der einen Code-Snapshot anzeigt, sobald eine dritte Person hinter dem Bildschirm erscheint, und ein „Ironman“-Modus, der eine IronMan-Maske auf jedem erkannten Menschen anzeigt.
